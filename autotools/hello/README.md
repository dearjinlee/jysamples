# Sequence

- `$ autoscan`
- edit 'configure.scan' and save it to 'configure.ac'
- `$ ./autogen.sh` <!-- hand-made script - run only once for installing missing scripts -->
- `$ autoreconf`
- `$ ./configure`
- `$ make`
