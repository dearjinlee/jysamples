# Creating New Gradle Builds

* https://guides.gradle.org/creating-new-gradle-builds/?_ga=2.107779714.1854664234.1531195753-2006220189.1531195753

## Initialize gradle project

```
gradle init
```

or

```
gradle init --dsl kotlin
```

## Run task

```
./gradlew copy
```

## Tasks

```
./gradlew tasks
```

## Build scan

```
./gradlew zip --scan
```

## Properties

```
./gradlew properties
```
