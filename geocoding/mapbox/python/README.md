# mapbox SDK Py #

* https://github.com/mapbox/mapbox-sdk-py


## Install ##

```
$ pip install mapbox
```

## Run ##

```
$ MAPBOX_ACCESS_TOKEN="mapbox.access.token" python main.py
```
